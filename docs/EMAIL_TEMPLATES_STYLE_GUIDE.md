# Промпт для бекендера: стили и верстка email-писем

Обнови вёрстку и стили всех email-писем (регистрация, подтверждение, OTP, напоминания и т.д.), чтобы дизайн совпадал с текущим оформлением сайта. Ниже — палитра, шрифты и список текстов для единообразия.

---

## 1. Стили сайта (Clinical Aura — текущая тема)

### Цвета
| Переменная | HEX | Использование |
|------------|-----|---------------|
| **Акцент (розовый)** | `#E8638B` | Кнопки, ссылки, бейджи, выделения |
| **Акцент hover** | `#D4507A` | Наведение на кнопки |
| **Вторичный (бирюзовый)** | `#5BB5A2` | Второстепенные акценты, успех |
| **Фон основной** | `#FFFFFF` | Фон страницы |
| **Фон вторичный** | `#F8F9FB` | Карточки, блоки |
| **Текст основной** | `#1A1D23` | Заголовки, основной текст |
| **Текст вторичный** | `#5F6B7A` | Подзаголовки, описание |
| **Текст приглушённый** | `#7E8A9A` | Подписи, хинты |
| **Граница** | `#E2E6EB` | Разделители, рамки |
| **Успех** | `#4ade80` | Сообщения об успехе |
| **Ошибка** | `#ef4444` | Сообщения об ошибке |
| **Предупреждение** | `#f59e0b` | Предупреждения |

### Шрифты
- **Заголовки:** Plus Jakarta Sans (или Inter как fallback)
- **Основной текст:** Inter
- **Fallback:** system-ui, sans-serif

Для email лучше использовать веб-безопасные шрифты: `Arial`, `Helvetica`, `sans-serif`. Или подключить Inter/Plus Jakarta Sans через Google Fonts.

### Типографика
- Заголовки H1: ~28–32px, жирный
- Заголовки H2: ~20–24px, жирный
- Основной текст: 16px, межстрочный интервал 1.5
- Мелкий текст: 14px
- Хинты: 12–13px, цвет `#7E8A9A`

### Стиль кнопок
- Фон: `#E8638B`
- Текст: белый, жирный
- Скругление: 8–12px
- Padding: 12–16px по вертикали, 24–32px по горизонтали

## 4. Рекомендации по HTML-письмам
1. Использовать table-based layout для совместимости с почтовыми клиентами
2. Inline CSS (многие клиенты игнорируют `<style>`)
3. Максимальная ширина контента: 600px
4. Логотип/брендинг в шапке — на акцентном фоне или белом
5. Кнопки — блоки с `background: #E8638B`, белый текст
6. Футер — приглушённый текст `#7E8A9A`, мелкий шрифт (12px)
7. Ссылки — цвет `#E8638B` (акцент)
