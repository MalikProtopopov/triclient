| 5 | **Telegram Bot API** — ограничения на добавление в каналы, смена условий API | Низкая | +2–4 часа backend | Бот — опциональная фича. При блокировке — fallback на ссылку-приглашение. Уведомления дублируются в email |
| 6 | **Scope creep** — расширение требований в процессе разработки | Средняя | +10–20% к общему бюджету | Модульная разработка с чёткой приёмкой каждого модуля. Новые требования оформляются отдельным модулем/этапом |
| 7 | **Производительность при росте базы** — деградация запросов при 5000+ врачей | Низкая | +8–16 часов на оптимизацию | Индексы заложены в БД-дизайне (db_v2.md). Для каталога — partial index по status='active'. Pagination cursor-based для больших списков |
| 8 | **Смена ставки/курса YooKassa** — изменение комиссии или условий | Низкая | Финансовый, не технический | Мониторинг условий. Абстракция платёжного провайдера позволяет быстрый переход |
