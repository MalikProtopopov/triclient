## Инструкция по созданию чекбоксов

### Google Sheets
1. Выделить `D2:D21`
2. Вставка → Флажок
3. Ячейка автоматически принимает `ИСТИНА` / `ЛОЖЬ`
4. Формулы `ЕСЛИ(D2=ИСТИНА;...)` работают напрямую

### Excel
1. Вкладка «Разработчик» → «Вставить» → «Элементы управления формы» → «Флажок»
2. ПКМ на флажке → «Формат объекта» → «Элемент управления» → «Связь с ячейкой: $D$2`
3. Повторить для каждой строки
4. Скрыть сам чекбокс-виджет или расположить поверх ячейки D

### Лист «Ставки» (B11 — Figma)
Аналогично: один чекбокс, связанный с ячейкой B11.
